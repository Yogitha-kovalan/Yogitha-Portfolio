Smart Course Manager

📌 Project Overview

Developed a Python-based course registration system using MySQL to manage studentenrollments and course records. Built a Tkinter GUI for students to register, view available courses, and track enrollmentsefficiently.

🔧 Features

  • Add new courses via a user-friendly GUI
  • View all courses stored in MySQL database

🖥️ Tech Stack

  • Python (Tkinter for GUI)
  • MySQL (via mysql-connector-python)

🚀 How to Run

  • Run database.sql to create the database and table
  • Install dependencies with pip install -r requirements.txt
  • Run the app with python app.py

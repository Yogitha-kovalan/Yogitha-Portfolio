Bomb & Landmine Detection and Diffusion System

The Bomb & Landmine Detection and Diffusion System is a smart robotic solution designed to detect explosive threats and toxic gas leaks in military and industrial environments. It combines advanced technologies such as metal detectors, gas sensors, GPS tracking, robotic arms, and real-time video monitoring to ensure safety and efficiency during high-risk operations. This project helps reduce the need for human intervention in dangerous zones by providing remote control and real-time alerts using IoT, making it a reliable tool for bomb squads, disaster response teams, and industrial safety units.

Main Components Used:

  • Arduino UNO
  • Metal Detector Sensor
  • Robotic Arm (Servo Motors)
  • IR Sensors, GPS Module, Bluetooth
  • Camera Module for live feed
  • Tracked chassis with L298N Motor Driver

Key Features:

  • Detects metal-based bombs and landmines
  • Shares live location using GPS
  • Controlled via Bluetooth
  • Uses robotic arm for safe bomb removal
  • LED indicators and buzzer alerts
